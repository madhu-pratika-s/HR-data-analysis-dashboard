# HR-Head Count-Data-Analysis (Using Power BI)
## Overview
The HR Headcount Analysis Dashboard is an interactive Power BI project designed to provide insights into employee distribution, diversity, and organizational structure. It helps HR departments monitor workforce trends, gender balance, and country-wise employee counts, enabling data-driven decision-making for effective workforce management.

## Dataset used
- <a href=https://github.com/madhu-pratika-s/HR-data-analysis-dashboard/blob/main/Employee%20Sample%20Data.xlsx">Dataset</a>

## Questions
- How many total employees are there in the company?
- How many male employees are there?
- How many female employees are there?
- What percentage of employees are male?
- What percentage of employees are female?
- Which business unit has the highest number of employees?
- Which business unit has the lowest number of employees?
- How many employees work in the United States?
- How many employees work in China?
- How many employees work in Brazil?
- What is the total number of employees in the Research & Development department?

## Process
- Collected employee data (headcount, gender, department, country, role) from Excel/CSV.
- Cleaned and formatted data using Power Query in Power BI.
- Created data model and DAX measures for key metrics.
- Designed interactive visuals for trend, gender, country, and department analysis.
- Added filters for date, business unit, country, and employee name.
- Tested dashboard for accuracy and interactivity.
- Exported visuals and documented the project.
